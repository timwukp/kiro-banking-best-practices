## Maintenance

These documents may become outdated as Kiro evolves. To refresh:

1. Visit the source URL listed above
2. Compare with the local copy for changes
3. Update the local file and snapshot date in this table

**Recommended refresh cadence:** Quarterly, or when upgrading Kiro versions.

**Note:** For the latest documentation, always check [kiro.dev/docs](https://kiro.dev/docs/). These local copies are provided as a convenience for teams in air-gapped or restricted environments.
